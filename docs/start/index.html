快速开始

低代码接口开发平台基于 Spring Boot 3、Python 3.12、amis 开发，服务端支持 Linux 4.18+ 内核，浏览器兼容 Chrome 109+。

部署​

  • 下载安装包 Gitee GitHub

  • 解压安装包

tar -xf crud-版本-打包日期-架构.tar.gz
  • 启动程序
cd crud
sh start.sh

登录​

系统启动后，使用浏览器打开低代码接口开发平台地址

  • 平台地址 http://ip:8889/
  • 默认账号 admin
  • 默认密码 111111

添加数据库​

在接口开发模块的操作栏中，点击"数据库管理"按钮进入数据库管理界面，点击"新增"按钮弹出新增数据库表单，填写目标数据库的连接信息，支持添加 Oracle、SQL Server、MySQL、PostgreSQL、SQLite 数据库。

数据库管理​

新增数据库​

数据库信息填写完成后，点击"测试"按钮验证配置信息是否正确，连接成功后将自动获取数据库版本，并回填至"数据库版本"字段。

接口开发​

支持通过编写 JSON 和 SQL 模板，直接对数据库表进行增删改查操作，并快速发布为接口。显著简化接口开发、测试及前端对接流程，同时兼容 amis 前端组件的数据格式。

相关概念​

平台统一规范接口 URI 目录命名规则。

  • 接口名称：/系统名称/模块名称/资源名称/操作名称
  • 接口 URI：/系统编号/模块编号/资源编号/操作编号
  • 接口地址：默认 8888 端口 http://ip:8888/接口 URI
端口参数对应 application.yml 文件中的 server.port 配置项，默认端口号为 8888

接口列表​

新增接口保存成功后，默认排在第一位，点击操作栏"测试"按钮，可对刚添加的接口进行测试操作，验证输入输出数据正确性。

新增接口​

点击”新增 JSON/SQL 接口”按钮，系统将弹出配置表单。请依次填写基本信息及 JSON/SQL 模板内容，确认信息无误后，点击“保存”完成创建，新创建的接口显示在接口列表第一行。

  • 基本信息
  • JSON 模板

接口测试​

新增接口保存成功后，系统会默认将其排列在列表第一位，方便您快速定位。您可以直接点击该接口操作栏中的“测试”按钮，立即对接口进行调试，并通过查看响应结果来验证输入输出数据的正确性。
